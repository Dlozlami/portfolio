# DLM Portfolio

Personal portfolio site — **Dlozi Lloyd Mthethwa**

Ecosystem Architect · Software Developer · Information Designer

🌐 **Live:** [portfolio-wrwi.onrender.com](https://portfolio-wrwi.onrender.com)

---

## Stack

- **Frontend:** React 19 + Vite
- **Database & Auth:** Supabase
- **Storage:** Supabase Storage (cv + resources buckets)
- **Hosting:** Render

## Features

- Home page with latest writing and recent work
- Blog — Articles and Project write-ups
- Full CV page pulled live from Supabase
- Admin panel — manage profile, work history, skills, certifications, files and posts
- Multimedia file manager with copy-URL workflow
- Single post editor for Articles, Projects and Resources

## Local setup

```bash
git clone https://github.com/Dlozlami/portfolio.git
cd portfolio
npm install
cp .env.example .env
# Add your Supabase URL and anon key to .env
npm run dev
```
